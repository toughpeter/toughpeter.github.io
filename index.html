我使用trello的方式

面對工作生活中複雜的工作與頹團隊合作，透過一些工具能夠協助我們更加聚焦。

我選擇專案管理工具的考量點主要有以下幾點：

  1. 團隊協作人員：我的專案人數為五人以下，人員功能也較為單純，當團隊人數越多、工作內容功能越不同時，通常就需要更多的權限管理與流程管理。
  2. 容易上手好學習 選擇工具的第一步，先了解自己的專案複雜度，功能不用多，選擇最適合自己專案的，導入時也可以降低團隊學習門檻。
  3. 與其他工具生態系搭配 團隊是否有使用其他工具，像是Google Drive、Slack、Dropbox、Github、GitLab等工具？工具間是否有外掛可以串連資訊？

使用 Hexo 撰寫文章

2016-08-04
hexo

Hexo 文章內容設定

Front-matter

Front-matter 是檔案最上方以 --- 分隔的區域，用於指定個別檔案的變數，舉例來說，我的文章標題是「使用 Hexo 撰寫文章與管理文章圖片」，發布文章時間是「2016-08-04 07:36:55」：

除了以上範例，Hexo有提供其他的設定方便你管理文章：

  • title：文章標題，不一定要跟文章檔名一樣，文章標題的設定不會影響文章的網址
  • date：檔案建立的日期，格式為「2016-08-04 10:17:27」
  • updated：檔案更新日期，格式為「2016-08-04 10:17:27」
  • comments：是否開啟文章留言的功能，預設為 true
  • permalink：設定覆蓋文章的網址
  • categories：分類
  • tags：標籤
